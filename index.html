Условия

Чтобы поставить условную конструкцию — используйте ключевые слова if, else, endif.

<div class="header">
  @{ if Recipient.Balance > 500 }
    <div>У тебя уже больше половины баллов!</div>
  @{ else }
    <div>У тебя еще меньше половины баллов! Поднажми!</div>
  @{ endif }
</div>

Циклы

Чтобы бахнуть обход по коллекции, используйте конструкцию for of.

<div class="header">
  У тебя в корзине:
  @{ for product of Order.Products }
    <div>product.DisplayName</div>
  @{ end }
</div>
