Условия

Чтобы поставить условную конструкцию — используйте ключевые слова if, else, endif.

@{ if Recipient.Balance > 500 }
У тебя уже больше половины баллов!
@{ else }
У тебя еще меньше половины баллов! Поднажми!
@{ endif }

Циклы

Чтобы бахнуть обход по коллекции, используйте конструкцию for of.

У тебя в корзине: @{ for product of Order.Products }
${ product.DisplayName }
@{ end }
